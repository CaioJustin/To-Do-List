# TO DO LIST

 uma to do list feito  que permitir criar tarefas e deletar desenvolvida em **html** **Css** e **javascript Dom** 

## Funcionalidade

- **Criar Tarefas**  e possivel criar tarefas dinamicamente
- **Deletar tarefas** - Cada tarefa possui um botão "Remover" para exclusão individual
- **Marcar como concluída** - Checkbox para indicar tarefas finalizadas (com estilo visual) fazendo 
- **Persistencia de dados** As tarefas são salvas no LocalStorage
- **Carregamento Automatica**  As tarefas continuam mesmo apos a pagina ser atualizadas
